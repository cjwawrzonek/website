I. Background

There are countless options out there to host a website including lots of free options for hosting static webpages. My previous iteration of this site that I made in college was simply a set of html files hosted in a public google drive folder paired with a godaddy domain, which forwarded trafic to an 'index.html' page. Pretty simple. Since then I've stumbled on tons of options for hosting static sites, even easiser than this. Scott Murray has a great overview of some of these options here.

For this newer iteration, I wanted a more interactive site for hosting my projects and blog, and maybe dive a little deeper into the details of web hosting and web development. So I threw together this site using some very simple and easily approachable tools.
